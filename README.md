# PHÒNG KHÁM MINI

Ứng dụng quản lý tiếp nhận bệnh nhân và kê đơn thuốc.

---

## Công nghệ

- HTML5
- CSS3
- JavaScript ES Module
- Vite
- LocalStorage
- Vitest
- Playwright
- GitHub Actions
- GitHub Pages

---

## Cài đặt

```bash
npm install
```

---

## Chạy môi trường phát triển

```bash
npm run dev
```

---

## Build

```bash
npm run build
```

---

## Preview

```bash
npm run preview
```

---

## Unit Test

```bash
npm run test:unit
```

---

## Business Test

```bash
npm run test:business
```

---

## Coverage

```bash
npm run test:coverage
```

---

## Playwright

```bash
npm run test:e2e
```

---

## Playwright UI

```bash
npm run test:e2e:ui
```

---

## Playwright Report

```bash
npm run test:e2e:report
```
